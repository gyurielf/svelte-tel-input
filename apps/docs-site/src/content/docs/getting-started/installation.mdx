## Prerequisites

- **Svelte 5** (peer dependency)
- **Node.js** >= 22
- **pnpm** >= 10 (or npm/yarn)

## Install

## Peer dependency

`svelte-tel-input` requires **Svelte 5** as a peer dependency. Make sure your project is running Svelte 5+.

## What's included

The package ships:

| Export | Description |
|--------|-------------|
| `TelInput` | The main phone input component |
| `countries` | Array of all supported countries with dial codes |
| `parsePhoneNumberWithError` | Re-export from `libphonenumber-js` |
| `ParseError` | Re-export from `libphonenumber-js` |
| `inputParser` | Utility to parse raw input strings |
| `isSelected` | Helper for country list selection |
| `parsePhoneInput` | Low-level phone number parser (returns `DetailedValue`; accepts a `Country` object) |

Types are available from `svelte-tel-input/types` and flag CSS from `svelte-tel-input/styles/flags.css`.
