# svelte-tel-input docs

Documentation site for [svelte-tel-input](https://github.com/gyurielf/svelte-tel-input), built with [Astro](https://astro.build) and [Starlight](https://starlight.astro.build).

## Commands

Run from the repo root:

```sh
pnpm dev:apps      # start dev server
pnpm build:apps    # production build
pnpm preview:apps  # preview production build
```

Or from this directory:

```sh
pnpm dev
pnpm build
pnpm preview
```
